Canvas

Canvas (it's already plural trust me) are dynamic elements in html that can edit or create a picture in html using a script and the tag <canvas>. They often
have some sort of user interaction.

Here's some example code:

<canvas id="myCanvas"></canvas>
<script>
var canvas=document.getElementById('myCanvas');
var ctx=canvas.getContext('2d');
ctx.fillStyle='#FF0000';
ctx.fillRect(0,0,80,80);
</script>

Of course, this only creates a non-interactive red square, but canvas can be used in many ways.
